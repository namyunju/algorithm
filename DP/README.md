## 동적 계획법

큰 문제에 대한 답을 얻기 위해 동일한 문제이지만 크기가 더 작은 문제들을 먼저 해결한 뒤, 그 결과들을 이용해 큰 문제를 비교적 간단하게 해결하는 기법

- 점화식 세우기
- 재귀함수 이용
    - 종료 조건으로 초기 조건을 설정
    - 그렇지 않은 경우 점화식 작성

## 메모이제이션 memoization

값을 기록하고 기록한 값을 참조하는 것

재귀함수로만 구현 시 시간이 오래 걸림. 동일 계산을 여러 번 반복

memo라는 배열을 이용해 중복 연산을 제거

방식
- 함수 도입부에 memo된 값을 반환
- memo되지 않은 경우, 반환해줘야 하는 값을 memo에 저장해준 뒤 해당 값을 반환

## Tabulation

재귀 함수 이외에 동적 계획법 문제 해결 방법 

for문을 이용. 순서대로 배열에 값을 채워나가는 방식 Tabulation

```
set dp = [0,0,0, ...]
dp[1] = 1
dp[2] = 1

for i = 3... i <= n:
    dp[i] = dp[i-1] + dp[i-2]
print(dp[n])

```

- memoization
    - 높은 수에서 낮은 수로 내려가는 탑다운 방식
- tabulation
    - 아래에서 값을 채워나가므로 바텀업 방식

이론적 시간복잡도는 두 방법 동일하나 탑다운은 함수 여러 번 실행해야 해서 바텀업이 약간 더 빠름
